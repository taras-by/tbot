# tbot

##Help:
    /list - participants list
    /add - add yourself or someone
    /rm - remove yourself or someone
    /reset - remove all
    /ping - turn to non-participants
    /start - help
    /help - help

##Examples:
     /add @smith
     /add My brother John
     /rm @smith
     /rm My brother John
     /rm 3

The last example is the removal of the third participant
